# Student Grade Calculator

## Project Overview
This project implements a simple **Student Grade Calculator** using Python.  
It calculates the average marks of a student based on 5 subjects and assigns a grade accordingly.

## Grade Criteria

| Average Marks | Grade |
|--------------|-------|
| ≥ 90         | A+    |
| ≥ 75         | A     |
| ≥ 60         | B     |
| ≥ 50         | C     |
| < 50         | Fail  |
